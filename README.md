An integration to parse [Lottie](https://airbnb.io/lottie) files and render them with [Vello](https://vello.dev).

## Run Demo (Native)

- Native

  ```shell
  cargo run -p demo
  ```

## Run Nursery (WASM)

- WASM (requires `cargo install trunk`)

  ```shell
  cd demo
  trunk serve
  ```
